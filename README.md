## üìÅ Project Files Overview

The repository is divided into two main categories of exercises:

### üîπ Project Problems (No1 - No10)

| File | Description | Key Concepts |
|------|-------------|--------------|
| `No1Projek.cpp` | **Find Maximum Value** - Finds the highest value among 10 student grades and counts how many students achieved that maximum score | Arrays, While Loop |
| `No2project.cpp` | **Reverse Sentence** - Reverses the order of words in a sentence (e.g., "I eat rice" ‚Üí "rice eat I") | Arrays, String Manipulation, Functions |
| `No3projek.cpp` | **Filtered Average Calculator** - Calculates the average of array elements excluding values between 5 (inclusive) and 7 (exclusive) | Arrays, For Loop, Conditional Statements |
| `No4project.cpp` | **Length Unit Converter** - Converts millimeters to kilometers, meters, centimeters, and millimeters | Functions, Modulus Operator |
| `No5projek.cpp` | **Money Change Calculator** - Calculates the denominations of Indonesian Rupiah currency for a given amount | Arrays, Functions, Greedy Algorithm |
| `No6project.cpp` | **Time Converter** - Converts seconds into hours, minutes, and seconds | Pointers, Functions |
| `No7Projek.cpp` | **Number to Words Converter** - Converts a number (1-99) into Indonesian words (Terbilang) | If-Else Statements, Modulus |
| `No8project.cpp` | **World Cup Team Data Manager** - A menu-driven program to input and display team names for a World Cup-style tournament | Arrays, Functions, Do-While Loop, Switch Case |
| `No9projek.cpp` | **Geometry Calculator** - Comprehensive calculator for 2D shape areas (8 shapes) and 3D shape volumes (8 shapes) | Switch Case, Do-While Loop, Multiple Functions |
| `No10project.cpp` | **Color Harmony Calculator** - Determines color notation (Primary/Secondary/Tertiary), color properties (Warm/Cool), and calculates harmonious color combinations | Functions, Switch Case, Color Theory |
